# 🍽️ Indonesian Food Calorie Estimator

Aplikasi AI yang mendeteksi makanan Indonesia dari foto dan menampilkan estimasi kalori + nutrisinya secara otomatis.

## Features
- 📷 Deteksi makanan dari foto
- 🔥 Estimasi kalori, protein, karbohidrat, lemak
- 📊 Daily calorie tracker
- 🍛 10 kelas makanan Indonesia

## Tech Stack
- MobileNetV2 (Transfer Learning)
- TensorFlow/Keras
- Streamlit
- Hugging Face Hub

## Dataset
[rzyuanda/indonesian_food_v2](https://huggingface.co/datasets/rzyuanda/indonesian_food_v2)
